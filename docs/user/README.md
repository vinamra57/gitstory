# User Documentation

To setup GitStory as a user, please head to these files in the following order:
1) [set-up-API_user.md](set-up-API_user.md): Documents how to create your own Gemini API key for GitStory, which is essential for this project
2) [user-guide](user-guide.md): Documents how to setup the project for users

*Once you have setup GitStory using the above steps,* for running GitStory, please head over to:
- [run-commands-guide.md](run-commands-guide.md): Documents how to run certain GitStory commands as a developer

To **build GitStory as a final product and use as a GitStory user**, please head over to the [build-guide.md](https://github.com/vinamra57/gitstory/blob/main/docs/dev/build-guide.md).
